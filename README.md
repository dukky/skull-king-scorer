# Skull King Scorer

A simple score tracker for the [Skull King](https://boardgamegeek.com/boardgame/150145/skull-king) card game.

**Try it:** https://skullking.dev.duk.im

## Features

- 2-6 players
- Track bids, tricks, and bonus points (pirate captures, Skull King capture)
- Automatic score calculation
- Game state persisted in localStorage
- Mobile-friendly, can be added to home screen

## Development

Examples use Bun, but npm/pnpm/yarn work too.

```bash
# Install dependencies
bun install

# Run dev server
bun run dev

# Build for production
bun run build
```

## Tech

- React
- Vite
- No external UI libraries
